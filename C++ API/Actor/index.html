    Actor

    class Actor

    The Actor class represents an Actor in the game's world.

    See
    Direction

    Variables

    Name Description
    tileSet The used TileSet for this Actor's sprites.
    tileSetSource The path to the TileSet file.
    tile The currently shown tile from the TileSet.
    position Current position in the world of this Actor.
    collisionRect Rectangle, representing the collision of this Actor.
    animations An array of the animations.
    currentAnimation A Direction enum, showing the current animation that is being played.

    Variable Details

    animations

    std::array<std::unique_ptr<std::vector<Vector2>>, 8> animations

    An array of the animations. Each std::vector represents the frames of an animation.

    collisionRect

    Rectangle collisionRect

    Rectangle, representing the collision of this Actor.

    currentAnimation

    Direction currentAnimation

    A Direction enum, showing the current animation that is being played.

    position

    Vector2 position

    Current position in the world of this Actor.

    tile

    AtlasTile tile

    The currently shown tile from the TileSet.

    tileSet

    std::unique_ptr<TileSet> tileSet

    The used TileSet for this Actor's sprites.

    tileSetSource

    std::string tileSetSource

    The path to the TileSet file.
